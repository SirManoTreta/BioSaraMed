BIOSARAMED - primeira versão

Arquivos principais:
- index.html -> página inicial dinâmica
- sobre.html -> apresentação
- pesquisas.html -> listagem das pesquisas
- pesquisa.html -> detalhe de cada pesquisa
- artigos.html -> listagem monocromática de artigos
- artigo.html -> leitura individual do artigo
- contato.html -> contatos e perfis acadêmicos

Como usar:
1. Abra a pasta em um editor como VS Code.
2. Rode com uma extensão de servidor local, como Live Server.
3. Edite os textos em js/data.js.
4. Ajuste cores e estilos em css/styles.css.

Ponto mais importante para personalizar:
- Substitua o conteúdo fictício de Sara BioMed pelos dados reais da sua irmã.
- Troque e-mail, redes e áreas de interesse.
- Depois, posso converter essa base para React ou Next.js.
